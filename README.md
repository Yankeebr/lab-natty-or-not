O ebook "Aprendendo IA, Easy mode " oferece dicas práticas para otimizar sua interação com ferramentas de IA. ## 🤖 Tecnologias Utilizadas
IA Generativa ChatGPT para roteirização e geração de imagens;
Ferramenta WPS Office para edição.

## 🧐 Processo de Criação
Utilizei o "ChatGPT" como auxiliar na criação do e-book. a edição final foi feita no WPS Office, uma ferramenta open-source para editar o texto.

## 🚀 Resultados
Este Ebook elucida muito, no que fazer e como melhorar, ao ultizar ferramentas de IA.
